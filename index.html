사용 방법
💙 안내 사항
리버스: 1999 전용 위키 생성 페이지입니다. 자유롭게 이용해주세요.
모든 문의는 트위터(X)의 @NA5A1999 계정의 디엠이나 익명 질문함으로 주세요.
사용 예시는 이 페이지를 확인해주세요.
보이저 드림주의 사용을 금지합니다.
📝 텍스트 서식
텍스트를 드래그한 후 상단 서식 툴바 버튼을 클릭하세요.
아무것도 선택하지 않고 버튼을 누르면 텍스트라는 플레이스홀더가 삽입됩니다.
· B (굵게) — 선택한 텍스트를 굵게 표시
· / (기울임) — 선택한 텍스트를 기울임체로, 회색으로 표시
· 키워드 — 스킬명·상태이상 등 파란색 강조
· 수치 — % 수치 등 주황색 강조
📌 각주 삽입
1. 본문 텍스트 칸에 글을 쓰세요.
2. 각주로 연결할 단어를 드래그해 선택하세요.
3. 서식 툴바의 각주 버튼을 클릭하세요.
4. 팝업창에 각주 이름을 입력하세요. (예: 1)
5. 선택한 단어 윗첨자로 [1]가 자동 삽입됩니다.
6. 좌측 탭에서 각주 섹션으로 이동하면 자동 추가된 항목이 보입니다.
7. 내용을 입력하면 마우스 오버 시 툴팁에 자동 반영됩니다.
💡 기타 팁
· 통찰 재료: 등급 선택 → 드롭다운에서 아이템 선택 → 이름×수량 자동 입력
· 마도술 타입 아이콘: 드롭다운에서 선택하면 이미지 자동 적용
· 계승 이미지: 추가 순서대로 자동 배정 (최대 3개)
· JSON 저장으로 작업 내용을 보존하고, 나중에 불러올 수 있습니다.
· HTML 내보내기로 완성된 위키 파일을 받을 수 있습니다.
